# Installation
We will install vim on linux<br>
__Installed on Fedora Linux!!!__

```vim 
$ sudo dnf update
$ sudo dnf install vim
```

__Installing vim but already on macos!__<br> 
https://macvim.org/


In order to find out the version of vim, go to vim itself

```vim 
$ vim 
```
